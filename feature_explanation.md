# Feature Explanation

estimated_days:
Planned duration of a task in days.

actual_days:
Actual number of days taken to complete the task.

priority:
Task importance on a scale of 1 (Low) to 5 (High).

dependencies:
Number of other tasks this task depends on.

team_size:
Number of team members assigned.

is_delayed:
Target variable.
1 = Task delayed
0 = Task completed on time
